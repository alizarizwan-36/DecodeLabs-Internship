General Knowledge Quiz

Description
This project is a simple General Knowledge Quiz written in Python. The program asks the user three questions, checks the answers, provides feedback, and displays the final score at the end.

Features
- Asks 3 general knowledge questions.
- Accepts user input using `input()`.
- Uses `.strip()` and `.lower()` to sanitize input.
- Uses `if-else` statements to check answers.
- Keeps track of the score using a variable.
- Displays the final score using an f-string.

 Example Output

text
1. What is the capital of France? paris
Correct!
Your final score is corect/total

 Concepts Used
- Variables
- User Input
- String Methods (`strip()` and `lower()`)
- Conditional Statements (`if-else`)
- Arithmetic Operators
- f-Strings
